# SpaCy Models

In this location provides different basis models to be used for a Imxis-Workflow instance.



### invoice-de-0.1.0

Basic model for international invoices. This model is based on spaCy V.3.0

	Entities:
	##########
	cdtr.iban 
	cdtr.bic  
	cdtr.name 
	invoice.toal
	invoice.number
	invoice.date
